# 09. 시연 시나리오

## 형식

5분 분량의 사전 녹화 영상 + 발표 자료. 라이브 시연은 하지 않는다 (실패 리스크 회피).

영상은 한 번에 모두 한 컷으로 찍지 않고, 각 단계를 별도로 촬영한 뒤 편집으로 연결한다. 베스트 컷 선택 가능.

## 영상 구성 (5분 기준)

### 0:00 ~ 0:30 — 소개

- 시스템 한 줄 정의: "RGBY 색상 물체 중 빨간 불량품을 검출하고 자동 분류·운반하는 스마트 팩토리 시스템"
- 구조 다이어그램 한 장 (3 레이어 아키텍처도)
- 본 영상에서 다룰 4가지: 자율주행, 검출/분류, 통합 흐름, 음성 인터페이스
- 화면 분할 또는 자막으로 시각화

### 0:30 ~ 1:30 — Stage 3 하이라이트 (자율주행)

가장 시각적 임팩트가 큰 장면을 먼저 배치.

촬영 컷:

1. SLAM으로 만든 맵 표시 (RViz 화면 캡처)
2. 터틀봇이 home에 대기 중인 모습
3. 음성 명령 "불량품 비워줘" — 자막으로 발화 표시
4. 터틀봇이 출발하는 모습 (전체 공간 보이는 카메라)
5. RViz 동시 화면 — 경로 계획과 위치 추적이 함께 보이도록
6. 적재장 도착
7. (시간 여유 시) 사람이 경로에 들어와 우회하는 장면
8. home 복귀

자막: "음성 명령으로 출발 → 자율 경로 계획 → 도착 → 복귀"

### 1:30 ~ 2:30 — Stage 1 (검출/분류)

촬영 컷:

1. 컨베이어 시작 버튼 클릭 (UI 화면)
2. 벨트가 동작하기 시작
3. RGBY 4색 물체를 차례로 올림
4. 화면 분할 — 컨베이어 전경 + 검출 박스 오버레이된 카메라 영상 + UI 차트
5. 빨간 물체 통과 → 서보가 불량 갈래로 회전 → 떨어지는 모습
6. 다른 색 물체 통과 → 서보가 정상 갈래로 → 다른 박스에 떨어짐
7. UI 차트 누적 카운트와 불량률이 갱신되는 모습

자막: "HSV 색공간 기반 빨간색 검출 → 서보 분류 → DB 기록 → UI 실시간 차트"

### 2:30 ~ 3:30 — 통합 시연

3개 단계를 자연스럽게 연결한 한 사이클.

촬영 컷:

1. 컨베이어 동작 중, 빨간 물체들이 검출되어 터틀봇 박스에 누적
2. 카운트 임계 도달 알림 (UI에 표시)
3. 자동 트리거로 터틀봇이 출발 (음성 없이, 자동)
4. 적재장 이동 → 도착 → 복귀
5. 박스가 비워지고 다시 컨베이어 동작
6. 한 사이클 완료

자막: "카운트 자동 트리거 → 자율 운반 → 복귀 → 다음 사이클"

### 3:30 ~ 4:30 — 음성 인터페이스 + UI

촬영 컷:

1. "지금 불량률 얼마야?" 발화
2. UI에 transcript와 응답 텍스트가 동시 표시
3. 스피커에서 응답 음성 재생 (자막 동시 표시)
4. "총 몇 개 검사했어?" 발화 → 응답
5. "터틀봇 어디 있어?" 발화 → 응답
6. UI의 음성 이력 페이지 보여주기 (DB에 누적된 모습)
7. 안전 STOP 버튼 클릭 시연 — 컨베이어가 즉시 멈추는 모습
8. "정지" 음성 발화로도 같은 동작 가능함을 보여줌

자막: "Whisper STT → llama.cpp 의도 추출 → 자연어 응답 → TTS"

### 4:30 ~ 5:00 — 결론

- 4차 Dobot: 시간 됐으면 시연 컷 짧게, 안 됐으면 컨셉도 한 장 + "향후 과제"
- 한계점 솔직히:
  - 빨간색 1개만 불량으로 처리 (확장은 코드 5줄)
  - 공간 크기 한정
  - 멀티 적재장은 안 다룸
- 발전 방향: YOLO 학습으로 검출 모델 교체, 멀티 적재장, 4차 도전
- 마지막에 두 개발자 이름 또는 팀명 자막

## 촬영 시 카메라 설정

- 1920x1080 30fps 이상
- 컨베이어 클로즈업과 전체 공간 광각 두 종류 미리 확보
- UI 화면은 OBS Studio로 별도 녹화 (해상도 보존)
- 음성 시연은 별도 마이크로 보이스 트랙 분리 녹음 권장

## 편집 가이드

- BGM은 잔잔한 무료 음원 (Pixabay, YouTube Audio Library)
- 자막은 한국어 + 핵심 영어 키워드 병기 (HSV, SLAM, Nav2 등)
- 너무 빠르게 자르지 않음 — 각 컷이 4초 이상 보이도록
- 화면 분할은 최대 3분할까지만 (보기 어려움)
- 영상 끝에 README/GitHub 링크 자막

## 사전 녹화 일정

- W4 D22-23: Stage 3 영상 본 촬영
- W4 D24: 음성 인터페이스 영상
- W4 D25: 편집 + 자막
